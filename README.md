# Licentra

**The Command Center for SaaS Licenses, Subscriptions & Spend**

Licentra is a modern SaaS management platform that helps businesses organize software licenses, track subscriptions, monitor renewal dates, and gain better visibility into SaaS spending through a centralized dashboard.

This project was developed to explore modern web development practices, responsive UI design, and scalable frontend architecture.

---

## ✨ Key Features

- Centralized SaaS License Management
- Subscription Tracking
- Renewal Monitoring
- Vendor Management
- SaaS Spend Visibility
- Responsive User Interface
- Modern Dashboard Experience
